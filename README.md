Pada praktikum RPL 2 pertemuan 1, saya mempelajari konsep desain perangkat lunak berbasis komponen, yaitu pendekatan pengembangan aplikasi dengan membagi sistem menjadi modul-modul yang dapat digunakan ulang. 
Selain itu, saya mempelajari berbagai software design pattern yang membantu menyelesaikan masalah umum dalam pengembangan aplikasi. 
Materi ini juga membahas prinsip pengembangan pada skala enterprise, di mana aplikasi harus dirancang agar mudah diperluas, dipelihara, serta mampu menangani kebutuhan bisnis yang kompleks.
